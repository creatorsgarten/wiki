To use, first go to <https://creatorsgarten.github.io/videos-cms/>, then open the folder where you cloned the [videos](https://github.com/creatorsgarten/videos) repository.

## Generating chapters with AI

To generate chapters you will need:

- Subtitle files for the current language (VTT format)
- OpenRouter account with some credits

Steps:

![](https://usercontent.creatorsgarten.org/c/v1773576800/644c35a6802c02345887f156/image_vrv0ea.webp)

1. In the video edit page, under **Chapters**, click **Generate Prompt**
2. Click **Copy** to copy the prompt
3. Click **Open Router** to go to OpenRouter (opens in new tab) with the recommended model pre-selected
4. Paste the prompt
5. Run the prompt
6. Copy the output
7. Back to the CMS, click **Edit Chapters**
8. Paste the contents
9. Click **Save** to save the chapters, closing the modal
10. Outside the modal, click **Save** to save the video metadata file

Each generation costs ฿1–฿2.
